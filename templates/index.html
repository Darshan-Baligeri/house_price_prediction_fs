About the Project

This web application uses Flask to provide predictions of house prices based on various property features, such as the number of rooms, area, and location. The machine learning model processes these inputs and returns an estimated property price.

With its simple form, users can easily enter property details and receive quick predictions. Whether you're buying, selling, or just curious about property values, our tool provides useful insights powered by cutting-edge AI technology.
